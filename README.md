# 💰 Expense Tracker

A simple and clean Expense Tracker web application built using **HTML, CSS, and JavaScript**.  
This project helps users track their daily expenses in an easy and minimal way.

---

## ✨ Features

- Add expense with description and amount
- View all expenses in a list
- Automatic balance calculation
- Delete expenses from list
- Real-time updates

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

---

## 🚀 How to Use

1. Open the project in browser
2. Enter expense description
3. Enter amount
4. Click **Add Expense**
5. View updated balance and expense list
6. Delete items if needed
